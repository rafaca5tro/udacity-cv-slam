> **Udacity Computer Vision Nanodegree** -- SLAM Implementation

Simultaneous Localization and Mapping (SLAM) implementation for a 2D robot world using sensor measurements and motion updates.

---

## Key Concepts

- Graph-based SLAM
- Omega and Xi matrices
- Sensor measurement integration
- Motion model updates
- 2D map reconstruction
